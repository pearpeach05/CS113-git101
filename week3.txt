
else-if คือการให้โปรแกรมเลือกทำตามเงื่อนไขที่เรากำหนดไว้ว่าจะได้ผลลัพท์เป็นอย่างไร
โดยการเขียนจะเริ่มจาก
if(เงื่อนไข){
    คำสั่ง ;
}

เมื่อตรงกับเงื่อนไขโปรแกรมจะเข้าไปทำคำสั่งใน if 
ถ้าไม่ตรงจะไปเช็คเงื่อนไขอื่นต่อใน else if หรือ if

การใช้ else-if
    if (เงื่อนไข){
        คำสั่ง;
    }
    else if (เงื่อนไข){
        คำสั่ง;
    }
    .
    .
    else{
        คำสั่ง
    }
While เป็นการทำคำสั่งซ้ำๆ จนกว่าจะไม่ตรงตามเงื่อนไขหรือมีคำสั่งให้ออกจากลูป
เช่น break คือการให้ออกจากลูปนั้น
    Continue คือประมวลผลต่อ

Switch case 
โดยที่คำสั่ง switch จะนำค่าของตัวแปรที่อยู่หลังคำสั่ง switch มาเปรียบเทียบกับค่าที่อยู่หลัง case แต่ละคำสั่ง ถ้าตรงกัน ก็จะทำสายงานที่อยู่ใน case นั้น ๆ

วิธีใช้ while loop
    while (เงื่อนไข){
        .....
    }
For ส่วนใหญ่จะใช้เมื่อรู้จำนวนรอบ
วิธีใช้ for(ค่าเริ่มต้น ;จบที่ ;การเพิ่มค่าหรือลบค่า){
       ....
    }
